# Algorithms

Classic algorithms and data structures implemented in Python — sorting, graphs, trees, search, cryptography, and quantum computing fundamentals.

## Structure

```
searching-sorting-indexing/
  01-benchmark-sorts/     # Sorting algorithm benchmarks and comparisons
  02-adaptive-search/     # Search algorithm implementations
  03-full-text-index/     # Text indexing and retrieval
  04-b-tree-kv-store/     # B-tree based key-value store

dynamic-programming-greedy/
  # DP and greedy algorithm implementations

advanced/
  data-structures/        # Advanced data structure implementations
  quantum/                # Quantum computing fundamentals
  rsa/                    # RSA encryption implementation
```

## Topics Covered

- **Sorting** — benchmark comparisons across sorting algorithms
- **Search** — adaptive search strategies and full-text indexing
- **Data Structures** — B-trees, key-value stores, and advanced structures
- **Dynamic Programming** — optimization problems and greedy approaches
- **Cryptography** — RSA encryption from scratch
- **Quantum** — quantum computing concepts in Python

## Stack

- Python 3
- Each module is self-contained with its own tests and documentation
